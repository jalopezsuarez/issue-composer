Why it makes GitHub better

Your tracker never leaves GitHub.

Issue Composer doesn't replace your issue tracker — it supercharges the one you already have. Everything it creates is a plain GitHub issue, label or comment.

  • Kanban without GitHub ProjectsBoard statuses are simple status: labels — the board works on any repo in seconds, with zero configuration.
  • Visible to your whole teamTeammates on github.com see the same issues, labels and comments — no second tool to adopt, no sync to break.
  • Issues that respect your codebaseThe AI reads your README and source files before writing, so issues cite real paths and symbols — not inventions.
  • Zero lock-inClose the tab or delete the app whenever you want — every issue, status and comment is already in your repository.

Everything in one page

A full issue workflow that fits in your pocket

No app to install, no accounts to create. Open the link, add your token and start shipping.

AI issue writer

Describe what you want in plain words. The LLM turns it into a clean, structured issue with a proper title and Markdown body.

Conversational assistant

A ChatGPT-style chat with real tools: it lists, searches and reads issues, digs into the code, creates issues, comments and moves statuses for you.

Drag-and-drop Kanban

A board on top of plain issue labels — no GitHub Projects to set up. Drag between columns, reorder within one, tap to open.

Reads your code

For features and bugs, the AI scans the README and the relevant files first, so every issue reflects what's already built and what isn't.

Threaded comments

Read and post comments — and let AI draft replies that take the issue and the whole thread into account.

Instant search

A toggleable search bar on both the board and the list, powered by the GitHub Search API — with sort order built into the bar.

Private by design

There's no server. Requests go straight from your browser to GitHub and your LLM. Credentials never leave your device.

Encrypted settings transfer

Move your setup between devices as an AES-256-GCM encrypted blob plus its key — copy both, paste both, done.

Light, dark & auto

A monochrome, glassy, GitHub-flavored interface that feels native on the phone and scales to the desktop — with a theme that can follow your system.

How it works

1

Open the app

Load the page on your phone or add it to the home screen as a web app.

2

Add your keys

Paste a GitHub token and your OpenAI-compatible LLM settings in Settings.

3

Compose with AI

Write a note, choose a mode, and let the AI draft a code-aware title and body.

4

Publish & manage

Publish to GitHub, move it on the board, reply in threads — or just ask the assistant.

No backend. No telemetry. No middleman.

Your tokens stay on your device

Issue Composer is a single static file. Every request goes directly from your browser to the GitHub API and to your LLM provider — nothing passes through anyone else's server. Your credentials live only in your browser, and settings travel between devices as an AES-256-GCM encrypted blob with its own key.

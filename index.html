📝 Issue Composer

Redacta issues de GitHub con ayuda de un LLM y publícalos directamente.

1 Configuración

Token de GitHub y proveedor LLM compatible con OpenAI.

2 Repositorio

Conecta para listar tus repos, o escribe owner/repo a mano.

3 Nota y análisis del código

Describe lo que quieres reportar. El LLM analizará el repo y redactará el issue.

4 Revisar y publicar

Edita el título y el cuerpo antes de publicar.

Issue Composer · HTML + CSS + JS vanilla, todo embebido · sin dependencias externas
